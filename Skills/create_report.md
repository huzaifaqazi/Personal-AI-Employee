# Skill: Create Report

## Purpose
Generate structured reports from data and information.

## When to Use
- Weekly/monthly status reports
- Project progress updates
- Data analysis summaries
- Performance reviews

## Process
1. Gather all relevant data
2. Organize by category/theme
3. Add metrics and statistics
4. Include visualizations (tables/charts)
5. Add conclusions and recommendations

## Output Format
```markdown
# [Report Title]

**Period:** [date range]
**Prepared:** [date]
**Status:** [Red/Yellow/Green]

## Executive Summary
[2-3 sentence overview]

## Metrics
| Metric | Target | Actual | Status |
|--------|--------|--------|--------|
| Item 1 | X | Y | ✅/⚠️/❌ |

## Highlights
- Achievement 1
- Achievement 2

## Challenges
- Issue 1
- Issue 2

## Next Steps
1. Action 1
2. Action 2

## Recommendations
- Recommendation 1
- Recommendation 2
```
