# Skill: Summarize Document

## Purpose
Extract key points and create concise summaries from long documents.

## When to Use
- Processing meeting notes
- Condensing research papers
- Creating executive summaries
- Extracting action items

## Process
1. Read the full document
2. Identify main topics and themes
3. Extract key points (3-5 bullets per section)
4. Highlight action items or decisions
5. Create summary in standard format

## Output Format
```markdown
# Document Summary

**Source:** [filename]
**Date:** [date]
**Type:** [meeting/report/research/etc]

## Key Points
- Point 1
- Point 2
- Point 3

## Action Items
- [ ] Action 1
- [ ] Action 2

## Decisions Made
- Decision 1
- Decision 2
```
