# Done Folder

Completed tasks are archived here.

## Organization
Files are stored with completion date prefix:
- Format: YYYY-MM-DD_[original_filename].md
- Example: 2026-02-16_TASK_market_research.md

## Retention Policy
- Keep current month's completed tasks
- Archive older tasks monthly
- Review quarterly for permanent deletion

## What's Stored
- Original task description
- Completion notes
- Output/deliverables
- Time stamps

## Searching Completed Tasks
Use grep or search tools to find past work:
```bash
grep -r "keyword" Done/
```
