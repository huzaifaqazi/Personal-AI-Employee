# Needs Action Folder

Tasks that require your approval will appear here.

## What You'll Find Here
- Tasks involving external communications
- Financial or sensitive operations
- File deletions or major changes
- Anything requiring human judgment

## How to Approve
1. Review the task and AI's proposed action
2. If approved: Add "APPROVED_" prefix and move to Inbox
3. If rejected: Add feedback and move back to Inbox
4. If needs changes: Edit the file and move back to Inbox

## Example Approval Process
```
Original: NEEDS_APPROVAL_send_email.md
After review: APPROVED_send_email.md
Then: Move to Inbox for processing
```
