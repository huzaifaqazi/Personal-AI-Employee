## Common Commands

### Check Status
View `Dashboard.md`

### Submit Task
1. Create file in Inbox/
2. Use proper naming: URGENT_/TASK_/INFO_
3. Fill in task details

### Approve Task
1. Review file in Needs_Action/
2. Add APPROVED_ prefix
3. Move to Inbox/
